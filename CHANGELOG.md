# Changelog

## [0.2.1-beta] - 2026-03-29

### 🎆 新機能 (New Features)
- **In-Windowダイアログ**: 長いファイルパスの確認時に画面中央に詳細ウィンドウを表示
- **汎用ポップアップシステム**: 今後も活用可能なInWindowDialogクラスを実装
- **プログレッシブダイアログ**: パスの長さに応じて自動でシンプル/詳細を切り替え

### 🌨️ UI/UX改善 (UI/UX Improvements)  
- **転送パス表示**: 「転送元: /path/to/source | | 転送先: /path/to/dest」形式で表示
- **ASCIIアート果線**: Unicode未対応環境でのフォールバック対応
- **センタリング**: ウィンドウのサイズに応じた自動中央配置
- **選択肢表示**: 「[*はい] [ いいえ]」形式で直観的な選択表示

### 🔧 技術的改善 (Technical)
- **モード管理**: 'in_window'モードを追加して入力処理を分離
- **レスポンシブサイズ**: 画面サイズに応じた動的ウィンドウサイズ計算
- **メモリ管理**: ウィンドウ終了時の適切なリソース解放

### 🐛 バグ修正 (Bug Fixes)
- **ダイアログ表示**: 長いファイル名で確認情報が画面外に表示される問題を解決

## [0.2.0-beta] - 2026-03-29

### 🎉 新機能 (New Features)
- **ファイル操作機能**: コピー(C)、削除(D)、移動(M)、リネーム(R)キーによる基本操作を実装
- **進捗表示システム**: Mode行でのリアルタイム操作進捗表示
- **確認ダイアログ**: 削除・移動・コピー時の安全確認システム
- **バックグラウンド処理**: UIをブロックしない非同期ファイル操作
- **編集モード**: リネーム時のインライン編集（Emacsキーバインド対応）

### 🌟 改善 (Improvements)
- **日本語対応強化**: Unicode East Asian Width規格に基づく全角文字の正確な表示幅計算
- **表示システム改善**: 動的カラムレイアウトによる表示崩れの解消
- **ホームディレクトリ起動**: 起動時のデフォルトディレクトリをホームディレクトリに変更
- **レスポンシブレイアウト**: 端末サイズに応じた最適なファイル表示

### 🐛 バグ修正 (Bug Fixes)
- **日本語フォルダ名**: テンプレート、ダウンロード等の日本語フォルダ名表示問題を修正
- **カラムずれ**: `<DIR>` カラムや拡張子カラムの位置ずれを修正
- **左右ペイン統一**: 左右ペインの表示不一致問題を解決

### 🔧 技術的改善 (Technical)
- **表示幅計算**: `get_display_width()` 関数による正確な文字幅計算
- **文字列処理**: `truncate_string_by_width()` による適切な文字列切り詰め
- **スレッド処理**: ファイル操作の非同期処理実装
- **エラーハンドリング**: ファイル操作エラーの適切な処理

## [0.1.0-beta] - 2026-03-28

### 🎉 初回リリース (Initial Release)
- **基本TUI**: 2ペインファイラーの基本実装
- **Emacsキーバインド**: Ctrl+P/N/A/B/E/F + TABによる操作
- **ファイル表示**: ディレクトリ/ファイル一覧、シンボリックリンク対応
- **色分けシステム**: ls --color=auto互換の色分け
- **プロジェクト構造**: モジュール分離による保守性向上
